


Angular Framework : core concepts
----------------------------------


    1- data-binding

        a- interpolation

            => to bind data from comp's class to comp's template

            syntax:

            in comp's template

                {{javascript-expression}}


    2. directives

        => buil-in function which direct how to render UI   

        a. structural

            => change structure of comp's template

            syntax:

            <element *directiveName="expression"/> 

            e.g

            <button *ngIf="boolean"></button>
